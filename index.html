Overlay Manager (ALPHA)

This is the open-source, alpha version of my biggest personal project. It's purpose is to manage the live streaming of Esports in a better way. A user will input the details of a series that is about to take place (team names, logo, and series length). Then, a second window will launch, acting as an overlay for the user to use on their stream. The overlay will be different depending on the game being streamed, in order to account for the in-game UI. I am currently working on the full version of this project, which will have an updated look as well as more features for the user to maximize the production quality of their livestream.
